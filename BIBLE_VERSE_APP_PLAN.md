## 1. 서비스 개요

### 한 줄 요약

듀오링고처럼 게임형으로 성경 말씀을 한/영 동시에 암송하는 앱 — 아이가 스스로 "또 하고 싶다"고 말하게 만드는 것이 핵심.

### 문제 정의

한국 기독교 가정에서 4-8세 자녀에게 성경 암송을 시키고 싶지만:

- 기존 방법(종이 카드, 부모가 직접 반복)은 **부모 시간이 많이** 들고
- 아이가 **금방 지루**해함
- 영어 암송까지 병행하고 싶지만 **한/영을 체계적으로 연결해주는 도구가 없음**
- 기존 앱(Remember Me, Bible Memory App 등)은 **영어권 성인 위주**라 한국 아이들이 쓰기엔 부적합

### 타겟 유저

| 구분 | 설명 |
|------|------|
| **주 사용자** | 미취학~초등 저학년 (4-8세) |
| **결제/관리자** | 기독교 가정의 부모 (주로 엄마) |
| **사용 맥락** | 가정 신앙교육, 주일학교 연계, 영어 학습 병행 |
| **확장 타겟** | 초등 전체 → 중고등학생 → 성인 (Phase 2 이후) |

### 핵심 가치 제안 (Value Proposition)

> "과학적으로 검증된 6가지 암기법을 게임으로 녹여,
> 아이가 한국어로 이해하고 영어로 암송하는
> 세계 유일의 이중언어 성경 암송 앱"

## 3. 학습 방법론 (연구 기반)

> 본 앱의 학습 시스템은 인지심리학 및 교육학 분야의 검증된 연구에 기반합니다.

### 3-1. 간격 반복 (Spaced Repetition)

- **근거**: Dunlosky et al. (2013) — 10가지 학습법 중 "높은 효용" 등급. PNAS(2019) Duolingo 데이터 기반 최적 알고리즘 연구.
- **적용**: 새 구절 학습 후 1일→3일→7일→14일→30일 자동 복습 스케줄. "오늘 복습할 구절"이 자동으로 제시됨.

### 3-2. 인출 연습 (Active Recall / Retrieval Practice)

- **근거**: Roediger & Karpicke (2006) — 인출 연습이 재읽기보다 1주 후 기억 유지율이 현저히 높음. Dunlosky et al. (2013) 유일한 "높은 효용" 등급. 초등학생 대상 연구(Karpicke et al., 2016)에서도 연령/능력과 무관하게 효과 확인.
- **주의**: 어린 아이에게는 비구조화된 자유 회상(free recall)보다 **가이드된 인출 연습**이 필요 (Blunt & Karpicke, 2014).
- **적용**: 난이도 점진적 상승 방식의 게임
  - Level 1: 단어 2개만 빈칸
  - Level 2: 절반 빈칸
  - Level 3: 첫 글자 힌트만
  - Level 4: 전체 빈칸 (완전 암송)

### 3-3. 이중 부호화 / 그림 연상법 (Dual Coding Theory)

- **근거**: Paivio (1971) — 언어+이미지 두 채널 동시 처리 시 기억 유지 대폭 향상. Pressley (1977) — 이미지가 단순 반복보다 어휘 회상 향상. Clark & Paivio (1991) — "그림 우월 효과" (picture superiority effect) 입증.
- **적용**: 각 구절의 핵심 키워드를 이모지/일러스트와 연결
  - 예: "태초에 하나님이 천지를 창조하시니라" (창 1:1)
  - → 🌅태초에 → 👑하나님이 → 🌍천지를 → ✨창조하시니라
  - 이모지 시퀀스를 보고 구절을 떠올리는 훈련

### 3-4. 노래/리듬 부호화 (Musical Encoding)

- **근거**: 에콰도르 아동 대상 외국어 학습 연구 — 노래로 배운 그룹이 말로만 배운 그룹 대비 정확한 암기, 발음, 번역 모두 우수했으며 6개월 후에도 효과 유지. 인도네시아 유치원 연구(Sutikno et al., 2024) — 노래가 기억력, 학습 동기, 창의성 모두 향상 (도파민/세로토닌/엔도르핀 분비).
- **적용**: 각 구절을 짧은 멜로디(Suno AI)로 제작. "노래로 먼저 만나기" 단계를 학습 플로우 첫 단계에 배치.

### 3-5. 청킹 (Chunking)

- **근거**: Miller (1956) — 작업 기억의 한계 7±2개. 의미 단위로 묶으면 한계 극복 가능.
- **적용**: 긴 구절을 한 번에 보여주지 않고 의미 단위로 쪼개서 하나씩 누적
  - Step 1: "태초에" 듣기 + 따라하기
  - Step 2: "태초에 / 하나님이" 연결
  - Step 3: "태초에 / 하나님이 / 천지를 창조하시니라" 전체

### 3-6. 동작 연결 (Total Physical Response / Gesture)

- **근거**: Asher (1960s) — 언어+신체 동작 결합 시 기억 유지 향상. 수학 학습 연구(2024) — 제스처 기반 교육이 장기 기억 유지에 더 효과적.
- **적용** (Phase 2): 키워드별 간단한 손동작을 애니메이션으로 보여주고 따라하기. 앱 카메라로 동작 인식도 가능 (향후).

## 4. 핵심 학습 플로우

> 하나의 구절을 외우는 전체 과정 (1회 세션: 약 5-7분)

### 4-1. 스텝 순서의 연구 근거

> **v1.2 변경**: 기존 "노래→그림→청크→게임→완료" 순서를 Mayer의 멀티미디어 학습 인지 이론에 근거하여 재배치하고, 기도 스텝을 추가했습니다.

**Mayer의 Cognitive Theory of Multimedia Learning (2001)**에 따르면, 학습자는 시각 채널과 청각 채널 두 개의 작업기억 통로를 사용하며, 각 채널의 용량은 제한적입니다. 최적의 학습을 위해 다음 원칙을 적용합니다:

| 원칙 | 설명 | 적용 |
|------|------|------|
| **사전 구조화 (Pre-training Principle)** | 핵심 개념의 구조를 먼저 보여주면 이후 정보를 처리할 스키마(틀)가 형성됨 | 그림 스텝을 가장 먼저 배치하여 구절의 의미 구조를 시각적으로 형성 |
| **멀티미디어 원칙 (Multimedia Principle)** | 텍스트만보다 텍스트+이미지가 효과적 | 모든 스텝에서 시각+청각 요소를 함께 제공 |
| **시간적 인접 원칙 (Temporal Contiguity)** | 관련 시각/청각 정보는 동시에 또는 가까이 제시해야 함 | 의미(그림) 이해 직후 노래(청각)를 제시하여 이중 부호화 동시 활성화 |

**4-8세 아동에게 특히 효과적인 이유**: Mayer의 연구에서 사전 지식이 적은 학습자(low prior knowledge)일수록 Pre-training 효과가 더 큽니다. 아이들은 성경 구절에 대한 사전 지식이 거의 없으므로, 의미 이해를 먼저 시키는 것이 효과가 큽니다.

**기도 스텝 추가 근거**:
- **Depth of Processing** (Craik & Lockhart, 1972) — 단순 암기보다 개인적 의미 부여가 깊은 처리를 만들어 장기기억 전환율이 높아짐
- **Self-Reference Effect** (Rogers, Kuiper & Kirker, 1977) — 정보를 자기 자신과 연결할 때 기억 효과가 2-3배 증가
- 기도는 구절을 "내 말"로 바꾸는 과정이라 자연스러운 정교화 시연(elaborative rehearsal)

### 4-2. 6단계 학습 플로우

```
① 그림으로 이해하기 (Dual Coding — Pre-training)
   └─ 이모지/일러스트 시퀀스로 구절 의미 시각화
   └─ 한국어로 의미 이해 확인
   └─ 역할: 의미 스키마 형성 → 이후 청각 정보의 앵커
   
        ↓

② 노래로 만나기 (Musical Encoding)
   └─ 구절이 담긴 짧은 노래를 듣고 따라 부르기
   └─ 한국어 버전 먼저 → 영어 버전
   └─ 역할: 이미 이해한 의미 위에 청각 부호화 얹기 (이중 부호화 동시 활성화)
   
        ↓

③ 청크로 쌓기 (Chunking)
   └─ 구절을 3-4개 청크로 분할
   └─ 하나씩 듣기 → 따라 말하기 → 연결하기
   └─ 한국어 → 영어 순서
   └─ 역할: 작업기억 단위로 분해하여 부담 감소
   
        ↓

④ 게임으로 외우기 (Active Recall) ← 핵심 게임 영역
   └─ Game 1: 빈칸 채우기 (단어 터치로 선택)
   └─ Game 2: 첫 글자 힌트 (나머지 떠올리기)
   └─ Game 3: 단어 순서 맞추기 (드래그 앤 드롭)
   └─ Game 4: 전체 암송 (음성 or 타이핑)
   └─ 난이도 자동 조절: 맞추면 올라가고, 틀리면 내려감
   └─ 역할: 능동적 인출 연습으로 기억 고정
   
        ↓

⑤ 기도로 새기기 (Depth of Processing + Self-Reference)
   └─ 구절 키워드를 활용한 짧은 기도문 (한/영)
   └─ 따라 읽기 또는 TTS로 듣기
   └─ 역할: 구절을 '내 이야기'로 전환 → 깊은 처리 + 장기기억 전환
   
        ↓

⑥ 오늘의 완료! (동기부여)
   └─ 별/하트/XP 획득
   └─ 캐릭터 경험치 증가
   └─ 연속 출석 스트릭 표시
   └─ "내일 다시 만나자!" 예고
   
        ↓

⑦ 내일 다시 만나기 (Spaced Repetition)
   └─ 자동 복습 스케줄에 따라 이전 구절 + 새 구절 조합
```

## 5. 기능 정의 (Phase별)

### 🟢 Phase 1: MVP (첫 출시)

**학습 핵심**
- 구절 10개 (창세기 중심, 한/영)
- 노래로 만나기 (구절당 짧은 멜로디 — Suno AI)
- 그림 연상 (이모지 시퀀스)
- 청크 학습 (의미 단위 분할 → 점진적 누적)
- 빈칸 채우기 게임
- 첫 글자 힌트 게임
- 단어 순서 맞추기 게임
- 한/영 TTS 오디오 (ElevenLabs)
- 한→영 학습 플로우 (한국어 이해 → 영어 암송)

**학습 관리**
- 30일 플랜 1개 (하루 1구절 + 복습)
- 간격 반복 스케줄 (자동)
- 진도 추적 대시보드 (오늘 했는지 / 며칠 연속 / 외운 구절 수)
- 출석 스트릭 (듀오링고 스타일)

**동기부여**
- 별/XP 획득 시스템
- 간단한 캐릭터 (레벨업 표시)
- 완료 축하 애니메이션

**기타**
- 모바일 우선 반응형 웹
- 부모용 간단 설정 (아이 이름, 학습 알림 시간)
- 푸시 알림 ("오늘의 말씀이 기다리고 있어요!")

### 🟡 Phase 2: 소프트 런칭 (피드백 반영 후)

- 구절 확장 (30개+, 주제별 컬렉션)
- 캐릭터 꾸미기 ("말씀 갑옷" 아이템 — 에베소서 6장 연계)
- 부모 대시보드 (아이 학습 현황, 외운 구절 목록)
- 동작 따라하기 (TPR — 키워드별 손동작 영상)
- 음성 녹음 + 비교 (아이가 직접 암송한 것 vs 원본)
- 복습 게임 추가 (이모지만 보고 전체 구절 말하기)

### 🔴 Phase 3: 정식 런칭

- 친구 연결 + 랭킹 대결
- 교회/그룹 기능 (담임 목사/교사가 커리큘럼 설정)
- 이벤트/대회 (교회 간 암송 대회)
- Bible Readers 연동 (암송한 구절이 나오는 스토리 읽기)
- 연령별 난이도 자동 조절 (4세 vs 8세 vs 12세)
- 성인 모드 (QT 연계, 묵상 저널)

## 10. MVP 구절 콘텐츠 (10개)

### 구절 목록 (개역한글 + WEB)

**1. 시편 56:3** — 테마: 두려울 때 😨→🙏→😊

| 한국어 (개역한글) | English (WEB) |
|---|---|
| 내가 두려워하는 날에는 주를 의지하리이다 | When I am afraid, I will put my trust in you. |

- 청크: "내가 두려워하는 날에는" / "주를 의지하리이다"
- 선정 이유: 가장 짧음. 아이가 무서울 때 바로 떠올릴 수 있는 실용 구절. MVP 첫 구절로 성공 경험 제공.

**2. 빌립보서 4:4** — 테마: 기쁨 🎉→😄→🔁→😄

|---|---|

- 청크: "주 안에서 항상 기뻐하라" / "내가 다시 말하노니" / "기뻐하라"
- 선정 이유: "기뻐하라" 반복 구조 → 노래/리듬 부호화에 최적.

**3. 시편 150:6** — 테마: 찬양 🌬️→🎵→🙌→🎉

**성경 암송 현재 상태**

1. 아이에게 성경 암송을 시키고 있나요? 시킨다면 어떤 방식으로?
2. 암송 시킬 때 가장 어려운 점은 뭔가요?
3. 암송 관련 앱이나 도구를 써본 적 있나요? 있다면 어땠나요?

**이중언어 니즈**

4. 영어로 성경 암송하는 것에 관심이 있나요?
5. 한국어로 먼저 이해하고 영어로 외우는 방식은 어떻게 생각하세요?

**제품 컨셉 반응**

6. 아이가 듀오링고처럼 게임으로 말씀을 외울 수 있다면 써보고 싶으세요?
7. 이런 서비스에 월 얼마까지 낼 수 있으세요?

### 인터뷰 핵심 포인트

- 질문보다 **듣기에 집중** — 예상 못한 인사이트가 나올 수 있음
- "좋을 것 같아요"보다 **구체적 행동**을 확인 — "실제로 다운받아서 써볼 의향이 있으세요?"

## 15. Bible Readers와의 관계

Bible Readers 프로젝트는 보류하되 폐기하지 않습니다.

**단기 (6개월)**: 암송 앱에 집중. Bible Readers는 현재 상태(2개 에피소드)로 유지.

**중기 (6-12개월)**: 암송 앱이 안정화되면, 암송한 구절이 등장하는 Bible Readers 스토리와 연동. 예: 창세기 1:1을 외운 아이 → "천지창조" 스토리 언락.

**장기**: 하나의 플랫폼 (MannaBooks)으로 통합. 암송 → 스토리 읽기 → AI 상상력 독서 → 토론 → 글쓰기의 4단계 비전 유지.
