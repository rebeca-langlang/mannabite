## 2. 음성 (TTS Audio)

### 규격
| 항목 | 규격 |
|------|------|
| 형식 | **MP3** (128kbps 이상) |
| 샘플레이트 | 44100Hz |
| 한국어 목소리 | 따뜻한 여성 (엄마 느낌), 천천히, 또박또박 |
| 영어 목소리 | 밝은 여성, 느린 속도, 발음 명확 |
| 추천 도구 | **ElevenLabs** (multilingual v2) 또는 **Typecast** (한국어 특화) |

### 파일명 규칙
```
public/audio/{verse-id}/{type}-{lang}.mp3
```

### 구절별 필요 음성 파일

각 구절마다 아래 파일들이 필요합니다:

| type | 설명 | 예시 텍스트 (psalm-56-3) |
|------|------|------------------------|
| `full-ko` | 전체 구절 한국어 | "내가 두려워하는 날에는 주를 의지하리이다" |
| `full-en` | 전체 구절 영어 | "When I am afraid, I will put my trust in you." |
| `chunk1-ko` | 청크 1 한국어 | "내가 두려워하는 날에는" |
| `chunk2-ko` | 청크 2 한국어 | "주를 의지하리이다" |
| `chunk1-en` | 청크 1 영어 | "When I am afraid," |
| `chunk2-en` | 청크 2 영어 | "I will put my trust in you." |
| `prayer-ko` | 기도문 한국어 | "하나님, 제가 무서울 때..." |
| `prayer-en` | 기도문 영어 | "God, when I am afraid..." |

> 청크 수는 구절마다 다름 (2-3개). 아래 전체 텍스트 참고.
